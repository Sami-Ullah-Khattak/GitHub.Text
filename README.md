# **GitHub**

- The platform to build,scale, and deliver secure software.    
- It provide a site to work together on a project from anywhere.  
- Code hosting platform,version control and collaboration.
***

## **Repository**

- Used to organize a project.  
- It contain anything the project needs,i.e file,folder,image videos,spreadsheets,datasets, and a Readme file.
***

## **Readme file**

- Written in plain text Markdown Language.       
- A file with information about project.
***

## **Repository Branch**

- Different versions of repository at same time.  
- By default it contain one branch(main).
***

## **Commit**
- Saved changes in the branch is called commit.   
- It contain a message of why the change was made.
***

## **Create New Repository**

1. Use **"+"** drop down menu in the uprigth corner of page.    
2. Select **"new repository"**.    
3. Type **"name" of repository"**.  
4. Write a short description in **"description box"**. 
5. Select "Add Readme file".   
6. Select repository type **"Public"**, **"Private"**.  
7. Click **"create repository"**.
***

## **Creating branch**  

1. Click **"branch"** button.  
2. In the next window click on **"New Branch"** button.     
3. Type branch name and click on **create branch** button.
